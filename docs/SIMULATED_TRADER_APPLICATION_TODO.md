# 模拟交易员应用级实施清单

更新时间: 2026-05-06

## 目标
- 构建“交易员独立思考系统”，对 AI 预测与推荐进行二次判断。
- 建立“评价-反馈-优化”闭环，持续提升训练、复盘、推荐质量。
- 达到可应用级别：可运行、可观测、可解释、可持续优化。

## 里程碑

### M1 思考系统最小闭环（本轮）
- [x] 新增交易员决策日志表（记录决策分、分项分、理由、最终动作）
- [x] 引擎接入决策分计算（预测分/推荐分/风险分/组合分）
- [x] 记录“采纳/拒绝”与触发原因
- [x] 新增思考日志 API
- [x] 新增 AI 评分卡 API
- [x] 后台页面展示思考日志 + 评分卡

### M2 自适应策略（下一轮）
- [x] 引入市场状态识别（趋势/震荡/高波动）
- [x] 动态阈值（买入阈值、单笔仓位、现金保留比例）
- [x] 持仓相关性约束（防止同质化过高）

### M3 优化闭环自动化（下一轮）
- [x] 失败交易分类器（模型误判/推荐偏置/风控不足/时机偏差）
- [x] 自动生成重训建议（资产、周期、优先级）
- [x] 自动生成复盘任务（按损失贡献排序）

### M4 生产可用性（下一轮）
- [x] API 权限与审计强化
- [x] 决策日志归档策略（性能与存储）
- [x] 页面告警与健康看板
- [x] 回归测试与验收脚本

## 验收标准
- 功能可用: 页面可见、接口可访问、数据可写入可读取。
- 决策可解释: 每次 buy/reject/sell 均有理由与分项得分。
- 评价可量化: AI 评分卡可输出采纳率、一致率、闭环损益质量。
- 可持续优化: 文档和代办清单持续更新，按里程碑推进。
